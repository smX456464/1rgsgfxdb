# 🎨 Pixiv 刮刮乐快捷跳转脚本

一键将 Pixiv 作品图片转换成炫酷的刮刮乐效果，**无需手动下载、粘贴链接**。  
支持**单图拼贴、单图纯色涂层、双图分离**三种模式，每种模式拥有独立的画笔、擦除力度等配置，并可记忆你的偏好。

---

## 📖 功能概述

| 功能 | 说明 |
|------|------|
| **📷 单图拼贴模式** | 适合已经拼好“涂层+底图”的图片，直接作为刮刮乐拼贴图使用。 |
| **🖌️ 单图纯色涂层模式** | 以**纯色图层**覆盖在作品图上，刮开即可看到原图，**完美解决非拼贴图无法刮开的问题**。纯色图层自动匹配原图尺寸，不会变形。 |
| **🖼️ 双图分离模式** | 分两步选择：先选一张做**底图**，再选另一张做**涂层**，自动生成双图层刮刮乐。 |
| **⚙️ 独立参数设置** | 每种模式拥有**独立的画笔大小、擦除力度**设置，互不干扰。 |
| **🎨 纯色涂层颜色** | 内置 **10 种颜色**（白、黑、灰、红、橙、黄、绿、蓝、靛、紫），可自由切换。 |
| **⌨️ 自定义快捷键** | 默认 `Ctrl + Alt + 右键`，支持 `Ctrl`、`Alt`、`Shift`、`Meta` 任意组合。 |
| **🌐 多刮刮乐站点支持** | 可添加多个刮刮乐服务地址（含本地 `file://` 路径），并能启用/禁用、切换。 |
| **📋 本地路径自动复制** | 当使用本地 HTML 文件路径时，自动复制完整链接到剪贴板，并给出提示，**避免浏览器阻止跳转**。 |
| **💾 配置持久化** | 所有设置自动保存，关闭浏览器后也不会丢失。 |

## 🚀 快速开始

### 1. 安装 Tampermonkey
- 下载浏览器扩展：[Tampermonkey 官网](https://www.tampermonkey.net/)
- 推荐使用 Chrome、Edge 或 Firefox。

### 2. 安装脚本
- 在 Greasy Fork 或 GitHub Release 中点击脚本链接，Tampermonkey 会弹出安装页面，确认即可。
- 或者直接新建脚本，粘贴完整代码。

### 3. 使用
1. 打开任意 Pixiv 作品页面（`https://www.pixiv.net/artworks/xxxxx`）。
2. 将鼠标悬停在作品图片上，按住 **`Ctrl + Alt`**（默认）不放，点击**鼠标右键**。
3. 在弹出的菜单中选择模式：
   - **📷 单图拼贴模式（当前图）**：直接使用当前图片。
   - **🖌️ 单图模式（纯色涂层 · 白色）**：纯色刮开看原图。
   - **🖼️ 双图模式**：先选底图，再选涂层，分步完成。
4. 脚本会自动生成链接并**新标签页打开**，或**复制到剪贴板**（本地路径时）。

## 🌐 自定义刮刮乐网站

脚本默认使用 `https://mirumo-scratch-card.pages.dev/`，你也可以添加自己的部署或本地文件。

1. 点击菜单 **🌐 载入网址**。
2. 在管理窗口中：
   - 勾选/取消勾选启用站点。
   - 点击 **添加** 输入完整网址（支持 `https://`、`http://`、`file:///`）。
   - 点击 **删除** 移除不需要的站点。
3. 列表**按顺序启用第一个**有效站点作为跳转目标。

> ⚠️ 本地 `file://` 路径无法自动打开新标签页（浏览器安全限制），脚本会自动复制链接并弹窗提示，你手动粘贴到地址栏即可。

## ⚠️ 已知问题

- **部分图片加载失败**：使用反代链接（pixiv.cat 等）时，某些 JPG 图片可能因编码差异或转存过程中的细微损坏，导致刮刮乐网站的 Canvas 无法解析，显示“图片加载失败”。  
  **暂无彻底自动解决方法。**  
  **建议**：优先使用“**单图纯色涂层**”模式，该模式下底图以普通 `<img>` 方式加载，兼容性极好；或手动下载图片后在刮刮乐页面上传。

- **本地文件无法自动跳转**：浏览器禁止从 `https://` 页面自动打开 `file://` 路径，脚本已自动转为复制链接提醒。

- **动图（Ugoira）不支持**：刮刮乐工具仅适用于静态图片。

## 🔧 链接生成规则

脚本会解析 Pixiv 图片的 `src` 属性，提取作品 ID、页码和扩展名，生成反代短链接：

- **第一张图**（页码为 1）：`https://pixiv.cat/作品ID.扩展名`  
  例如：`https://pixiv.cat/143949732.jpg`
- **其他页码**：`https://pixiv.cat/作品ID-页码.扩展名`  
  例如：`https://pixiv.cat/143949732-3.jpg`

之后拼接刮刮乐网站的基础 URL 和相应参数（`mode`、`brush`、`maxpull` 等）生成最终跳转地址。

## 🛠️ 技术细节

- **纯色涂层生成**：通过 `<canvas>` 动态创建与原图等大的纯色 PNG，以 Data URL 形式传入刮刮乐页面，**完全不受跨域限制**。
- **本地路径检测**：`openOrCopy` 函数判断 `file://` 或 `localhost` 链接，调用 `GM_setClipboard` 或 `navigator.clipboard.writeText` 复制，并回退到 `prompt` 兜底。
- **图片尺寸获取**：使用 `img.naturalWidth / naturalHeight` 获取原图尺寸，确保纯色涂层与作品图一致，避免拉伸。
